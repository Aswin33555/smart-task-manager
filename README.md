# 🧠 Smart Task Manager

A modern and professional Task Manager application built using React.js and Tailwind CSS.

This application allows users to create, update, complete, and delete tasks with persistent local storage support.

Tasks are automatically saved in the browser using LocalStorage, so data remains even after refreshing or closing the browser.

---

# 🚀 Features

- ✅ Add Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ☑️ Mark Tasks as Completed
- 💾 Auto Save using LocalStorage
- 📊 Task Statistics Dashboard
- 🌙 Modern Dark UI
- 📱 Responsive Design
- ⚡ Fast Performance using Vite

---

# 🛠️ Technologies Used

- React.js
- Tailwind CSS
- Vite
- LocalStorage API
